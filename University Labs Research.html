Why Build a Dedicated Lab?

Core Infrastructure Setup

Centralize your lab with a robust hypervisor. Proxmox VE is the recommended standard as it securely hosts 80% of lab workloads for free.

🖥️ Hypervisor Comparison

Proxmox VE (Recommended)

Free (AGPL). KVM VMs + LXC containers. Built-in web UI, ZFS support. Best for all-in-one lab hosting.

XCP-ng

Free (GPLv2). Xen-based. Best if GPU PCI passthrough is critical or migrating from Citrix.

Local Virt (Student Laptops)

VMware Workstation Pro (Free for personal), VirtualBox (Cross-platform), UTM (Mac M1/M2).

⚙️ Physical Cluster Options

Essential for teaching distributed systems (K8s, Hadoop).

Implementation Roadmap

A phased approach allows budget justification through demonstrated outcomes. Don't build everything at once.

1

Phase 1: Foundation (Months 1–2)

Budget: ~$3,500

  • Procure and set up primary Proxmox server.
  • Deploy core VMs (Ubuntu, Windows, Kali).
  • Install GNS3 Server, JupyterHub, VS Code Server.
  • Register for AWS Academy & Azure Dev Tools.
2

Phase 2: Expansion (Months 3–4)

Budget: ~$5,000

  • Build Raspberry Pi cluster & deploy Kubernetes (K3s).
  • Set up Docker Swarm, Kafka, Hadoop on Proxmox.
  • Add Cyber lab VLANs with vulnerable VMs.
  • Activate Budget Tier student subscriptions (THM, HTB).
3

Phase 3: Advanced (Months 5–6)

Budget: ~$7,000+

  • Add GPU server for ML/AI labs.
  • Deploy OpenStack (DevStack/MicroStack).
  • Set up Cisco CML for networking prep.
  • Integrate LMS (Moodle) with lab portal.
